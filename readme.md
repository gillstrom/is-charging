# is-charging

> Find out if a computer is charging


## Install

```
$ npm install is-charging
```


## Usage

```js
const isCharging = require('is-charging');

isCharging().then(result => {
	console.log(result);
	//=> true
});
```


## API

### isCharging()

Returns a promise that resolves in to a `Boolean`.
